# PythonTermProject [미세먼지 주의보]

각 지역별 미세먼지 농도를 확인하여 검색된 위치에 "매우나쁨~좋음" 등을 확인할수 있다.

---------
구현 예정 내용
---------
 1. pyqt 5를 통한 GUI 제작
 2. Open API를 통한 위치별 미세먼지 농도를 받아오기
 3. Gmail 을 통한 현재 미세먼지 농도 등의 값을 전송하기
 4. C/C++ 함수 혹은 라이브러리 연동
 5. distutils 모듈을 활용한 개발 패키지 배포
